# 🧪 Estratégia de Testes

## Objetivo

Definir como os testes serão conduzidos durante o projeto.

---

## Estratégia adotada

Serão realizados testes manuais utilizando casos de teste previamente elaborados.

Também serão executados testes exploratórios para identificar comportamentos inesperados da aplicação.

---

## Tipos de testes

- Testes Funcionais
- Testes Positivos
- Testes Exploratórios

---

## Evidências

Durante a execução serão registradas capturas de tela para comprovação dos resultados obtidos.

---

## Registro de defeitos

Todos os defeitos encontrados serão registrados seguindo um padrão contendo:

- Identificador
- Título
- Descrição
- Passos para reprodução
- Resultado esperado
- Resultado obtido
- Severidade
- Prioridade
