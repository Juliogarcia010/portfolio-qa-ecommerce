# Plano de Testes

## Objetivo
Planejar os testes da aplicação de e-commerce.

## Escopo
- Carrinho de compras
- Produtos relacionados a games

## Funcionalidades fora do escopo
Não serão testadas funcionalidades inexistentes no projeto.

## Ambiente de testes
- Windows 11

- Microsoft Edge

- Notebook

## Tipos de testes realizados
- Testes Funcionais

- Testes Exploratórios

- Testes Positivos

## Ferramentas utilizadas
- Cursor

- GitHub

- Git

- VS Code

- Jira

- Word

- Markdown

## Critérios de entrada
- Sistema funcionando.

- Ambiente disponível.

- Casos de teste elaborados.

## Critérios de saída
- Todos os casos executados.

- Bugs registrados.

- Evidências anexadas.
