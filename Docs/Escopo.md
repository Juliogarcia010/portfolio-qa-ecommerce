# 📋 Escopo dos Testes

## Objetivo

Definir quais funcionalidades da aplicação fazem parte dos testes deste projeto.

---

## Funcionalidades contempladas

- Listagem de produtos
- Carrinho de compras
- Atualização das quantidades
- Remoção de produtos
- Finalização da compra

---

## Funcionalidades fora do escopo

- Integração com meios de pagamento reais
- Cupons de desconto

---

## Observações

Este documento poderá ser atualizado conforme novas funcionalidades forem adicionadas ao projeto.
