# 🐞 Registro de Bugs

Nesta pasta estão documentados todos os defeitos encontrados durante a execução dos testes.

Cada bug contém:

- Identificador
- Descrição
- Passos para reprodução
- Resultado esperado
- Resultado obtido
- Severidade
- Prioridade
